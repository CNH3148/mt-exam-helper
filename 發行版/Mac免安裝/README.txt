====================================================
      MT Exam Prep - Release Version 2.0 (Portable)
                     User Manual
====================================================

【 中文簡介 】
這是一個隨插即用的「免安裝版」醫檢師刷題系統。
它包含了所有的題庫資料與獨立的執行環境，您可以直接放在隨身碟中帶到任何 Windows 電腦（如圖書館公用電腦）上執行，不需要安裝任何軟體！

【 如何啟動 】
1. 雙擊本資料夾中的「Start_App.bat」。
2. 會彈出一個黑色的終端機視窗（這是背景伺服器，請勿關閉）。
3. 系統會自動幫您打開瀏覽器並進入系統，即可開始刷題！

【 如何關閉 】
1. 刷題結束後，關閉瀏覽器分頁。
2. 回到黑色的終端機視窗，點擊右上角的「X」關閉即可。

【 備份與同步進度 】
您的所有刷題進度、錯題紀錄都會自動儲存在以下路徑：
👉 `app\saves\` 
如果您想把隨身碟的進度同步回家裡的電腦，只需要把 `saves` 資料夾裡的 `.json` 檔案複製覆蓋即可！

====================================================

[ English Introduction ]
This is a portable, plug-and-play version of the MT Exam Prep system.
This folder contains all the core components, databases, web files, and a standalone Python runtime environment. You can run it directly on any Windows computer without installing anything!

[ How to Start ]
1. Double-click the "Start_App.bat" file in this folder to run it.
2. A black terminal window will pop up (this is the backend server, please keep it open).
3. The system will automatically open your default web browser and load the app. You can now start studying!

[ How to Close ]
1. When you are done, close the browser tab.
2. Go back to the black terminal window and click the "X" in the top right corner to close it.

[ Backup and Sync Progress ]
Your study progress, wrong questions, topic notes, and Anki cards are all safely stored in the following path:
👉 `app\saves\` 

If you want to sync your progress between your USB drive and your home computer:
Simply copy and replace the `.json` files inside the `saves` folder!
