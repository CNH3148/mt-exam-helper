# 國考刷題系統 (Mac 免安裝版)

這份檔案夾包含了能夠在 Mac 上直接執行的免安裝版本。

## 🚀 第一次使用：解除蘋果安全隔離 (Gatekeeper)

從網路上（如 GitHub）下載的 Mac 應用程式，預設會被蘋果的 Gatekeeper 機制標記為「隔離狀態」。如果您直接雙擊執行，可能會看到**「無法打開，因為無法驗證開發者」**或**「檔案已損壞」**的錯誤訊息。

這時請按照以下步驟解除隔離：

1. 打開 Mac 的「**終端機 (Terminal)**」（可以按 `Command + 空白鍵` 搜尋 Terminal）。
2. 在終端機輸入以下指令（**注意 `xattr -cr` 後面有一個空白鍵**）：
   ```bash
   xattr -cr 
   ```
3. **將解壓縮後的 `Mac免安裝` 資料夾（或是解壓縮出的那個資料夾），直接用滑鼠拖曳到終端機視窗內**，路徑會自動補上。
4. 按下 **Enter** 執行。
5. 這樣就完成解除隔離了！這個步驟只需做一次。

*(備註：如果不想用終端機，您也可以嘗試對著 `Start_App.command` 點擊「右鍵」 -> 選擇「打開」，在跳出的警告視窗中選擇「強制打開」。)*

---

## 🏃 日常啟動方式

1. 雙擊執行 `Start_App.command`。
2. 背景會自動啟動伺服器，並自動幫您打開瀏覽器進入 `http://127.0.0.1:8080`。
3. 開始盡情刷題！

> **⚠️ 注意事項**
> 當您正在刷題與儲存進度時，請**絕對不要**關閉那個背後運作的終端機黑色視窗！
> 結束學習時，直接關閉該終端機視窗即可安全停止伺服器。

## 💾 存檔與備份

所有的作答紀錄、進度、標籤等個人資料，都會即時寫入 `app/saves/` 資料夾內。
如果要備份紀錄，或是轉移到別台 Mac，只要把整個 `Mac免安裝` 資料夾（或單獨 `saves` 目錄）拷貝走即可。
