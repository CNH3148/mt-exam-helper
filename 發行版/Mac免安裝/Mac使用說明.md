# 國考刷題系統 (Mac 免安裝版)

這是一個隨插即用的「免安裝版」醫檢師刷題系統。它包含了所有的題庫資料與獨立的執行環境，您可以直接放在隨身碟中帶到任何 Mac 電腦上執行，完全不需要安裝任何軟體或 Python 環境！

---

## 🚀 如何啟動

### 狀況 A：您是使用「一鍵下載指令」安裝的
如果您在終端機輸入了 GitHub 首頁提供的 `curl` 自動下載指令，那麼系統**已經自動為您解除了蘋果的安全隔離**！
1. 直接雙擊執行資料夾中的 `Start_App.command`。
2. 背景會自動啟動伺服器，並幫您打開瀏覽器進入 `http://127.0.0.1:8080`，即可開始刷題！

### 狀況 B：您是「手動去 GitHub 網頁下載」ZIP 檔的
從網路上手動下載的 Mac 應用程式，預設會被蘋果的 Gatekeeper 機制標記為「隔離狀態」。如果您直接雙擊執行，可能會看到**「無法打開，因為無法驗證開發者」**的錯誤訊息。請按照以下步驟解除隔離（只需做一次）：

1. 打開 Mac 的「**終端機 (Terminal)**」（可以按 `Command + 空白鍵` 搜尋 Terminal）。
2. 在終端機輸入以下指令（**注意 `xattr -cr` 後面要留一個空白鍵**）：
   ```bash
   xattr -cr 
   ```
3. **將整個 `醫檢師刷題系統_Mac` 資料夾，直接用滑鼠拖曳到終端機視窗內**，路徑會自動補上。
4. 按下 **Enter** 執行。
5. 解除隔離完成！以後只要雙擊 `Start_App.command` 就能啟動了。
*(備註：如果不想用終端機，您也可以嘗試對著 `Start_App.command` 點擊「右鍵」 -> 選擇「打開」，在跳出的警告視窗中選擇「強制打開」。)*

---

## 🛑 如何關閉系統

1. 刷題結束後，直接關閉瀏覽器分頁。
2. 回到背後運作的**黑色終端機視窗**，點擊左上角的紅色按鈕關閉視窗，即可安全停止伺服器。

> **⚠️ 注意事項**
> 當您正在刷題與儲存進度時，請**絕對不要**關閉那個背後運作的終端機黑色視窗，否則系統會斷線！

---

## 💾 備份與同步進度

所有的作答紀錄、進度、標籤與錯題本資料，都會即時寫入以下路徑：
👉 `app/saves/` 

如果您想把隨身碟的進度同步回家裡的電腦，只需要把 `saves` 資料夾裡的 `.json` 檔案拷貝並覆蓋即可！
如果您要備份紀錄，也可以直接把整個資料夾備份起來。
